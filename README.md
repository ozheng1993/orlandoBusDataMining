# orlandoBusDataMining
## Installation
1. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
2. Clone this repository
3. Run python
    ```bash
    python3 lynxBus.py 
    ``` 
## Data format
   1. bus/*csv
      [
       'index'
       'fields.type', 
       'heading', 
       'id', 
       'lastStop',
       'lastUpdate',
       'lat',
       'lon',
       'name',
       'route'
        ]
